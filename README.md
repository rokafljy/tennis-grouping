# 🎾 테니스 복식 대진표 만들기

테니스 정기모임에서 **참석자 이름만 넣으면** 코트별로 골고루 게임을 배정해 주는
복식(2:2) 대진표 자동 생성기입니다. 시합용 토너먼트가 아니라, 모두가 비슷한 횟수로
즐겁게 게임할 수 있게 미리 짜 두는 **친선 대진표**를 목표로 합니다.

## 이런 상황을 위해 만들었어요

- 복식 경기(한 코트당 4명, 2:2)
- 보통 2~3코트를 빌려서 20명 안팎이 함께 하는 모임
- 한 경기는 30분 이내(테니스 한 세트)
- 모임 시간 1~2시간 → 자동으로 2~4라운드로 나눠 배정

## 특징

- **공정한 경기 수** — 모든 참석자의 경기 수 편차가 최대 1 이내가 되도록 배정
- **다양한 파트너/상대** — 같은 사람과 반복해서 팀/상대가 되지 않도록 최소화
- **자동 휴식 배분** — 인원이 코트 수보다 많으면 라운드마다 골고루 돌아가며 휴식
- **모바일 친화 + 오프라인** — 코트에서 휴대폰으로 바로 사용, 서버 전송 없음
- **결과 공유** — 단톡방에 붙여넣을 수 있는 텍스트 복사 / 인쇄 지원
- **다시 섞기** — 마음에 안 들면 버튼 한 번으로 새 조합 생성

## 사용법

브라우저에서 `index.html`을 열기만 하면 됩니다. 빌드 도구가 필요 없습니다.

1. 참석자 이름을 한 줄에 한 명씩(또는 쉼표로 구분) 입력
2. 코트 수, 모임 시간(또는 라운드 수), 한 경기 시간 설정
3. **대진표 만들기** 클릭 → 라운드별 코트 배정과 휴식 인원이 표시됩니다

로컬에서 서버로 띄우려면:

```bash
npx http-server -p 8080 .
# 또는
python3 -m http.server 8080
```

## 대진표 배정 규칙

우선순위에 따라 배정합니다.

1. **경기 수 균등** (가장 중요) — 많이 뛴 사람이 먼저 휴식하도록 조정
2. **파트너 다양성** — 이미 팀이었던 조합은 피함
3. **상대 다양성** — 이미 상대였던 조합은 피함

매 라운드 뛰는 인원을 코트에 배정할 때, 여러 후보 조합을 만들어 위 규칙에 대한
"비용"이 가장 낮은 조합을 선택합니다. 같은 입력이면 항상 같은 결과가 나오며,
**다시 섞기**로 다른 조합을 얻을 수 있습니다.

## 구성

| 파일 | 역할 |
| --- | --- |
| `index.html` | 화면 구조 |
| `styles.css` | 스타일 (다크/라이트 자동 대응) |
| `app.js` | 입력 처리·렌더링·복사/인쇄 등 UI 로직 |
| `scheduler.js` | 대진표 배정 알고리즘 (브라우저·Node 공용) |
| `scheduler.test.js` | 알고리즘 검증 스크립트 |

## 테스트

```bash
node scheduler.test.js
```

여러 인원/코트/라운드 조합에서 경기 수 편차가 1 이하인지, 같은 입력이 같은 결과를
내는지(결정성) 검증합니다.
